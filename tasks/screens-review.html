§4 · Cross-cutting notes  ·  §5 · Consistency walk

ReconnectAny phase: returning facilitator/participant lands on current phase view with exact prior state (role, group, scribe, submissions, cast votes). No rejoin screen.
Control barEvery facilitator screen: phase name + Advance; disable guards mirror state-machine exit guards T2a–T2c.
PresenterFullscreen, zero interactivity, no sign-in; never links a person to an answer/selection/vote.
ParticipantPortrait-first, reflows to centered column on landscape laptop; one primary action per screen.
AnonymityOnly final votes are truly anonymous (I14 — no voter↔vote link ever). Quiz answers and value selections are known per participant inside the session — selections must be, to optimize group formation — but no screen ever shows who picked/selected what. Tallies displayed counts-only.
i18nde+en for every visible string; language-carrying: values, quiz content, learning texts, labels, workshop record. Animal names localized. Participant action texts free text, untranslated.
WalkEvery T1–T23 + T17a covered by an element or matrix note · 22/22 commands reachable · SPEC.md screens table honored.
