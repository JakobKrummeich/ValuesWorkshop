# Clean-Room Provenance

This repository is a from-scratch rebuild of a workshop platform concept the
author previously implemented in an employment context.

## Why a rebuild

The prior implementation is legally unavailable for reuse (§69b UrhG —
employer ownership of software created within the scope of employment
duties; additionally it contains employer branding).

## Rules (binding for all contributors, human and agent)

1. **No access to the old repository, its code, assets, configs, or design
   tokens — ever.** Not for "reference", not for "comparison".
2. Everything here is written from memory of *requirements and behavior*
   (facts and ideas, which are not copyrightable), never from old source.
3. Visual identity (palette, typography, naming) is deliberately new.
4. Architecture decisions are made fresh and documented in `SPEC.md`.

## Evidence trail

- First commit of this repository is this file plus `SPEC.md`, written
  before any code existed — the spec was reconstructed from memory in a
  recorded session, then implemented forward from it.
- Full git history documents that all code derives from `SPEC.md`.
